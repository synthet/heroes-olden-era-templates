# Templates

Deployable Olden Era `.rmg.json` files (plus matching `.png` previews when present).

| Folder | Contents |
|--------|----------|
| `official/` | 57 native OE templates (clone these for new work) |
| `max8/` | 12 experimental 240×240 eight-player variants |
| `h3-port/` | 20 scaffolded H3 HotA tournament ports |

Copy needed files into the game's `StreamingAssets\map_templates\` — see [`../docs/install.md`](../docs/install.md).

Descriptions: [`../docs/template-catalog.md`](../docs/template-catalog.md).

Validate: `python tools/validate_rmg.py templates/`
