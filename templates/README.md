# Templates

Deployable Olden Era `.rmg.json` files (plus matching `.png` previews when present).

| Folder | Contents |
|--------|----------|
| `max8/` | Experimental 240×240 eight-player variants |
| `h3-port/` | Scaffolded H3 HotA tournament ports |

Copy needed files into the game's `StreamingAssets\map_templates\` — see [`../docs/install.md`](../docs/install.md).

Stock/official OE templates are **not** redistributed here (upstream license). Author those from your own game install or permitted community sources.
